Setup Guide

1
Create Facebook Business Portfolio
Go to business.facebook.com → Create Account → Name it 2EasyMarketing
2
Create a Meta Developer App
Go to developers.facebook.com/apps → Create App → Business type
3
Generate Your Access Token
Inside your App → Tools → Graph API Explorer → Generate token with ads_management & ads_read permissions
4
Add Payment to Ad Account
Business Manager → Ad Accounts → Add a credit card (Meta charges this for ad spend)
5
Paste Credentials Above & Test
Enter your Token, Ad Account ID, and Page ID above → click Test Connection → you're live
