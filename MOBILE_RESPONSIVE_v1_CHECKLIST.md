# Mobile Responsive Pass v1

## What was added

- Mobile viewport + theme color
- Floating mobile menu button
- Sidebar drawer on phones
- Backdrop close behavior
- Touch-friendly buttons and inputs
- One-column layout on phones
- Two-column layout on tablets
- Responsive Growth OS page
- Responsive Pricing page
- Responsive Plan Access page
- Table overflow protection
- iPhone input zoom prevention
- Safe-area support for newer iPhones

## What to test on a phone

1. Open the dashboard.
2. Tap the menu button in the top-left.
3. Open Pricing.
4. Open Plan Access.
5. Open Growth OS.
6. Generate One Post Everywhere.
7. Check the Publish Queue.
8. Try the Readiness Checker.
9. Rotate the phone.
10. Confirm no page has sideways scrolling.

## Truthful note

This improves mobile responsiveness, but the final mobile experience should still be tested on a real iPhone and Android phone after deployment.
